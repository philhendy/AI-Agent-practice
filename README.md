# Mountain Booking

A React-based booking platform tailored for the outdoor sporting community. Users can search and book flights and accommodation for mountain walking and cross-country mountain biking adventures. The site features a colorful green-themed design inspired by mountain landscapes and allows booking both services separately or together.

## Features

- Flight booking form (origin, destination, dates, passengers).
- Accommodation booking form (location, dates, guests).
- Combined view showing both booking options side-by-side.
- Responsive layout with nature-inspired green palette.

## Getting Started

### Install

```bash
cd mountain-booking
npm install
```

### Development

```bash
npm start
```

Opens the app in development mode at [http://localhost:3000](http://localhost:3000).

### Build for production

```bash
npm run build
```

### Notes

Optimized for simple client-side booking flows; forms currently log data to console and show alerts. Replace with API calls as needed.
